# Browser-Agent-Kit Roadmap

This document outlines the planned evolution of browser-agent-kit over the coming quarters. We prioritize features that enhance agent autonomy while maintaining safety and controllability.

# Q3 2026 (Active Development)

# Core Capabilities
- Basic DOM scraping and interaction (Complete)
      * Session Persistence:** Save/restore agent contexts across browser restarts
      * Multi-Tab Orchestration:** Coordinate actions across multiple browser tabs simultaneously
      * Human-in-the-loop Verification:** Pause critical actions requiring manual approval

# Developer Experience
- TypeScript API exports
      * Visual Debugger:** UI overlay to inspect agent decision trees live
      * Plugin Architecture:** Allow custom actions via SDK extension points


# Q4 2026 (Exploration Phase)

# Advanced Features
* Self-Correction Loops:** Agents detect and recover from failed actions autonomously
* Tool Integration:** Native connectors for GitHub, Slack, and Notium APIs
* Enterprise Mode:** RBAC controls and audit logging for team deployments

# Performance
- Optimize memory footprint for long-running sessions (>24h uptime)
- Reduce CPU overhead during idle periods
